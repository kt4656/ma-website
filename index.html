WHY "LEFT ARM"

なぜ、「右腕」ではなく「左腕」なのか

「社長の右腕」という言葉は、よく使われます。
会社を動かす実行力、社長の代わりに現場を回す存在です。

しかし、M&Aや事業承継という人生で一度きりの局面では、「実行力」だけでは足りません。
必要なのは、本音で語り合い、最善の判断を一緒に考えてくれる存在。
表舞台に立つ右腕ではなく、社長の隣で静かに支え、守る「左腕」です。

右腕は、あなたや社員の方。左腕は、私が務めます。

月額顧問料で稼ぐコンサルの場合、不安を煽りM&Aを長引かせることで報酬を増やそうとするケースもあります。

→ SOLUTION

半年一括の顧問料のため、案件を長引かせるインセンティブが構造的にありません。M&A仲介もお任せいただけるため、中立的な判断が可能です。

FAQ

よくあるご質問

Q1まだ売ると決めていませんが、相談できますか？

+

はい、できます。むしろ「そもそも売るかどうかを最後まで一緒に考える」ことが本サービスの根幹です。仲介と違い、売却を前提としない顧問契約ですので、現状の整理や企業価値の把握だけでも十分にご活用いただけます。

Q2仲介会社との違いは何ですか？

+

仲介会社は売り手と買い手の双方から手数料を受け取る「両手取引」が一般的です。顧問契約のみの場合、当社は売り手であるあなたとだけ契約し、買い手からは一切報酬を受け取りません。構造的に利益相反が発生しないモデルです。

Q3半年120万円+消費税の顧問料では何をしてもらえますか？

+

企業価値の簡易算定、承継の選択肢整理、買い手候補の分析、DD対策、交渉戦略の策定、仲介会社の評価など、士業固有の業務以外すべてサポートします。電話・LINE・WEB・対面、いつでもご相談可能です。

Q4顧問料の半年120万円+消費税以外に費用はかかりますか？

+

基本的にかかりませんが、遠方のお客様で対面でのご面談依頼がある場合は、交通費などをご相談させていただく場合もあります。

Q5売却が成立した場合、顧問料はどうなりますか？

+

顧問のみご依頼の場合は、特段成功報酬などはありません。M&A仲介業もご依頼いただく場合は、成功報酬から過去の顧問料は差し引いた金額をクロージング日にお支払いいただきます。

Q6売却しなかった場合、顧問料は返金されますか？

+

返金はございません。ただし、企業価値の把握や承継戦略の整理は、売却の有無にかかわらず経営に活かせる資産です。

Q7買い手側の顧問も務めていると聞きましたが、利益相反にはなりませんか？

+

同一案件で売り手・買い手の双方に就くことは基本的にありません。仮にお客様が私の顧問先に売却をされたい場合は、利益相反にならないようFA（片側アドバイザー）にて対応します。

Q8対応可能な企業規模・業種はありますか？

+

売上1億円〜50億円程度の中小企業が中心ですが、基本的に業種の制限はありません。製造業、建設業、サービス業、IT、介護など全ての業種に対応可能です。

Q9秘密は守られますか？社員や取引先に知られませんか？

+

秘密厳守を徹底しています。NDA（秘密保持契約）を締結し、社員・取引先・金融機関などへの情報漏洩が起きないよう細心の注意を払います。

Q10顧問契約は半年で終了ですか？延長できますか？

+

継続をご希望の場合は更新可能です。売却のタイミングは案件によって異なりますので、状況に合わせて柔軟に対応します。

Q11このサービスは売却検討されている人は誰でも使えますか？

+

申し訳ございませんが、全ての方にご対応できるわけではありません。Q12のような方とはお付き合いが難しいため、お断りさせていただく場合もございます。

Q12このサービスが合わない人は？

+

私と性格が合わない方。建設的なお話合いができない方。お堅いコンサルのようなイメージでサポートをしてほしい方。会社や従業員やステークホルダーのことを考えず、ただ単に売り抜けたいという気持ちの方。

Q13このサービスが合う人は？

+

M&Aに限らず、何かに疑問を持った際に気軽に相談できる左腕のような人を探している方。人間味がある人に最後まで確実にサポートしてほしい方。

Q14契約までの流れは？

+

初回は30分程度、WEBまたは対面にてお話をお伺いします。ご関心をお持ちいただけましたら契約内容をご案内し、ご納得のうえでご契約となります。

Q15顧問料が半年単位での支払いの理由は？

+

月額報酬の場合、M&Aを少しでも先延ばしにした方が顧問料をもらえてしまうという構造を排除するためです。

また、事業承継は半年から数年単位でかかるものですが、月額報酬型だと短期で結果を出すような提案をしてしまい、長期的なアドバイスを軽視してしまうと考えています。あくまでも長期的に社長の事業承継・人生のサポートができたらと思っております。

売るかどうかは、話してから決めればいい。
